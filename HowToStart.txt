Install
  a. node
  b. npm

  Steps

1. Configure Git Local Repo
2. In terminal, navigate to repo directory.
3.  'npm init'
        & enter necessary values
4. Configure .gitignore for node
5.  'npm install express --save'
    'npm install reload --save'
    'npm install ejs --save'
        These will install dependancy. Package.json will be updated.
6. initial commit to repo
